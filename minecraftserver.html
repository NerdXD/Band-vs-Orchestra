To join our Minecraft server, check out the instructions below!



MAKE SURE Grayson Ops you (Level 1) or you wont be able to join!

Java

1: Click on the button that says "Multiplayer" and then click "Add Server"

2: Type whatever you want in the name box and type "play.minehut.gg"

3: Once you are in the Minehut Hub, type the command "/join BandVSOrch"



Bedrock

1: Click play and then navigate over to the "Servers" tab

2: Scroll all the way down and click "Add Server"

Type whatever you want in the name box and put bedrock.minehut.com. Keep the port at 19132

Once you are in the Minehut Hub, type the command "/join BandVSOrch"
